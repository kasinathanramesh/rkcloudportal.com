Enumerate web server information using Nmap Scripring Engine (NSE)

Enumerate the directories used by web serversandweb applications, in
the terminal window. Type nmap -sV -script=http-enum <target
website> and press Enter.

The next step is to discover the hostnames that resolve the targeted
domain.

In the terminal window, type nmap -script hostmap-bfk -script-args
hostmap-bfk.prefix=hostmap- www.goodshopping.com and press
Enter.

Perform an HTTP trace on the targeted domain. In the terminal window,
type nmap --script http-trace -d www.goodshopping.com and press
Enter.

Now, check whether Web Application Firewall is configured on the target
host or domain. In the terminal window, type nmap -pBO --script httpwaf-
detect www.goodshopping.com and press Enter.

Uniscan web server fingerprinting in Parrot Security

Turn on the Windows Server 2016 virtual machirre and log in with the
credentials Administrator and pa$$word.

Start WAMPServer on the Windows Server 2016 virtual machine.
Double-dick the WAMPServer shortcut icon on Desktop to start the server

In the parot terminal window, type uniscan -h and hit Enter to display the
uniscan help options.

In the terminal window, type uniscan -u http:/110.10.10.16:8080/CEH -
q and hit Enter to start scanning for directories.

Note: Scroll to analyze the complete output of the scan. It should take
approximately 10 minutes for the scan to fmish.

Now, run uniscan using two options together. Here -w and -e are used
together to enable the ftie check (robots.txt and sitemap.xml ftie). In the
terminal window, type uniscan -u http://1 0.1 0.1 0.16:8080/CEH -we and
hit Enter to start the scan.

Now, use the dynamic testing option by giving the command -d. Type
uniscan -u http://1 0.1 0.1 0.16:8080/CEH -d and hit Enter to start a
dynamic scan on the web server.

Crack FTP credentials using a Dictionary Attack

Here, we will firstly find the open FTP port using Nmap, and then perform a
dictionary attack using the THC Hydra tool.

Note: Here, we will use a sample password @e (Passwords.txt) containing a list
of passwords to crack the FTP credentials on the target machine.

Perform an Nmap scan on the target machine (Windows 10) to check if
the FTP port is open.

In the parrot terminal window, type nmap -p21 [IP Address of Windows
1 0], and press Enter.

Open a new terminal and jump to the root directory. Now, type hydraL
/root/Wordlists/Usernames.txt -P /root/Wordlists/Passwords.txt
ftp://[IP Address of Windows 10] and press Enter.

N ote: The IP address ofWindows 10 in this lab exercise is 10.10.10.10. This
IP address might vary in your lab environment.

Type mkdir Hacked and press Enter to remotely create a directory named
